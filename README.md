# 반응형 자리 배치 프로그램 
사용자 설정에 따라 좌석을 생성하고, 애니메이션 효과와 함께 무작위로 자리를 배정해주는 웹 기반 도구입니다. 
교실, 세미나, 소모임 등에서 공정하고 재미있게 자리를 배치할 때 유용합니다. 

## Features 
- **실시간 애니메이션**: 추첨 시작 시 숫자가 돌아가는 시각적 효과를 통해 긴장감을 더합니다. 
- **유연한 레이아웃 설정**: 콤마로 구분된 입력을 통해 각 분단(세로줄)마다 원하는 좌석 수를 자유롭게 설정할 수 있습니다. 
- **결번 제외 기능**: 전입/전출이나 부재중인 번호를 제외하고 추첨할 수 있습니다. 
- **반응형 디자인**: 모바일과 데스크톱 환경 모두에 최적화된 UI를 제공합니다. 
- **데이터 저장**: 추첨 완료 후 최종 배치 결과를 CSV 파일로 다운로드할 수 있습니다. 
- **다크 모드 스타일**: 눈이 편안한 다크 블루 톤의 그라데이션 UI를 적용했습니다. 

## Usage 
1. **설정 입력**: 
   - **전체 인원 수**: 학급이나 그룹의 총 인원을 입력합니다. 
   - **결번 번호**: 제외해야 할 번호가 있다면 콤마로 구분하여 입력합니다.
   - **세로줄 배치**: 각 줄에 배치될 좌석 수를 입력합니다. (예: `5,5,5,6,5,5`는 총 6분단을 만듭니다.)  
2. **시작**: `설정 완료 및 시작` 버튼을 눌러 교실 배치도를 생성합니다. 
3. **추첨**: `추첨 시작` 버튼을 클릭하면 모든 좌석에서 숫자가 무작위로 회전하며, 잠시 후 왼쪽 앞자리부터 순차적으로 번호가 고정됩니다. 
4. **저장**: 모든 추첨이 완료되면 나타나는 `엑셀 저장` 버튼을 눌러 결과를 저장합니다.
